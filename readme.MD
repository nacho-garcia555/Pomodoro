🎯 FocusFlow — Pomodoro con Ruido Ambiental

Una aplicación web de técnica Pomodoro diseñada para maximizar tu concentración con temporizador, sonidos ambientales generados por código y frases motivacionales. Todo en un diseño oscuro, inmersivo y con animaciones fluidas.

HTML5CSS3JavaScript

✨ Características principales

Temporizador Pomodoro: 50 minutos de enfoque y 10 minutos de descanso.
Anillo de progreso animado: Representación visual del tiempo restante con transición de colores (Ámbar para enfoque, Teal para descanso).
Mezclador de ruido ambiental: 5 sonidos generados en tiempo real con Web Audio API (Lluvia, Tormenta, Fogata, Viento, Noche). Sin archivos de audio externos.
Frases motivacionales: Rotan automáticamente cada 20 segundos con animaciones de desvanecimiento.
Transiciones de modo: Cambio visual completo (colores, fondo, partículas) al pasar de Enfoque a Descanso.
Animaciones inmersivas: Partículas flotantes en el fondo con conexiones, efecto de "respiración" en el glow del temporizador y celebración con partículas al completar una fase.

Controles de teclado: Espacio (Play/Pausa), R (Reiniciar), S (Saltar fase).
Accesibilidad: Etiquetas ARIA, navegación por teclado y soporte para prefers-reduced-motion.

🛠️ Tecnologías utilizadas
HTML5 & CSS3 (Variables CSS, Flexbox, Animaciones)
JavaScript (Vanilla, Web Audio API, Canvas API)
Tailwind CSS (vía CDN)
Font Awesome (Iconos)
Google Fonts (Space Grotesk & DM Sans)
